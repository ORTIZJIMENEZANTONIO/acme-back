# acme-back
## Pre requirements:
- node v14.17.4
- npm v6.14.14 or yarn v1.22.19

## Run project
- Download repository
- You can use "yarn install" or "npm i"
- Run in command line "npm run dev"
- The project will be run in port 3000

## Oprtional
- In "docs" folder can use the collection in postman to check the url test
